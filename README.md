This is a simple landing page to signup for Amarelo's, my girlfriend's vinyl printing business, newsletter and marketing communications. <br>

I am using Mailchimp's API to capture the email address of form submissions and adding them to the Mailchimp account. These contacts will then be used to send targeted offers and marketing communications. <br>

This project taught me how to:
-Create API keys in multiple platforms <br>
-Validate form submissions <br>
-Hide API credentials in GitHub repositories safely <br>
-Read API documentation and develop solutions <br>
-Create a server to validad GET and POST submissions <br>
-Use the server to redirect a user to a success or failure landing page based on API response <br>

You can view this project live here: https://amarelo.onrender.com/
